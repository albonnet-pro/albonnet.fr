# Grille tarifaire — Alexis Bonnet, Développeur Web Freelance

> Référence interne. Fourchettes établies pour un profil junior BUT MMI (0–2 ans d'expérience), marché français.

## Forfaits projet

### Sites vitrine

| Périmètre | Fourchette |
|---|---|
| Site vitrine simple (template adapté, peu de personnalisation) | 500 – 1 200 € |
| Site vitrine sur-mesure (design Figma + intégration) | 1 500 – 3 000 € |
| Site vitrine + back-office personnalisé (gestion autonome du contenu) | 2 500 – 4 500 € |

> Le back-office permet au client de modifier textes, images et sections sans toucher au code — idéal pour les indépendants et TPE qui veulent garder la main sur leur site.

### Applications web personnalisées

## Règles de cadrage

- **Toujours formaliser le périmètre** avant de chiffrer — les dépassements non anticipés sont la première cause de travail non rentable.
- **Ne pas descendre sous 500 €** pour un site vitrine sur-mesure, même "pour se faire connaître" : en dessous, le temps passé n'est pas couvert.
- **Tarif "lancement"** acceptable sur les premiers clients, mais le poser comme exceptionnel et limité dans le temps, pas comme valeur de référence.
- **Réviser à la hausse tous les 12–18 mois** selon le portfolio et les retours clients.
